## 과정에서 만난 문제들과 해결책

### 1. GCC 버전 불일치 (핵심 문제)
- **증상**: `nvcc` 컴파일 시 `#error -- unsupported GNU version! gcc versions later than 11 are not supported!`
- **원인**: 시스템 GCC가 13.3.0인데 CUDA 11.8은 GCC ≤11만 지원
- **해결**: `conda install -c conda-forge gcc=11 gxx=11` 로 conda 환경 내에 GCC 11 격리 설치

### 2. crypt.h 헤더 누락
- **증상**: `fatal error: crypt.h: No such file or directory`
- **원인**: Ubuntu 24.04에서 libxcrypt 분리됨
- **해결**: `conda install -c conda-forge libxcrypt`

### 3. WSL2 링커 경로 문제 (/lib64 없음)
- **증상**: `ld: cannot find /lib64/libm.so.6`, `cannot find /lib64/libmvec.so.1`
- **원인**: WSL2는 `/lib64`가 없고 `/lib/x86_64-linux-gnu/`에 라이브러리 위치
- **해결**: sudo 없이 LDFLAGS로 경로 직접 지정:
  ```bash
  LDFLAGS="-L/lib/x86_64-linux-gnu -Wl,-rpath,/lib/x86_64-linux-gnu" \
  TORCH_CUDA_ARCH_LIST="8.9" \
  CPATH="/home/jskim/anaconda3/envs/3dgs/include:$CPATH" \
  conda run -n 3dgs pip install submodules/diff-gaussian-rasterization
  ```

### 4. COLMAP faiss 심볼 충돌
- **증상**: `colmap: symbol lookup error: undefined symbol: _ZN5faiss11write_indexEPKNS_5IndexEP8_IO_FILEi`
- **원인**: conda-forge COLMAP이 CUDA 12.9 빌드만 제공 → CUDA 11.8 환경과 faiss 버전 충돌
- **해결**: CUDA 빌드 대신 CPU 전용 빌드 설치:
  ```bash
  conda remove -n 3dgs colmap faiss faiss-cpu libfaiss
  conda install -n 3dgs -c conda-forge "colmap=3.13.0=cpu*"
  ```
  - COLMAP의 CUDA 가속(특징점 매칭)은 포기하지만 SfM 파이프라인 자체는 정상 작동
  - 3DGS 학습은 PyTorch CUDA로 하므로 실질적 성능 영향 없음

## 빌드 재현 명령어 (처음부터 다시 할 경우)

```bash
# 1. conda 환경 생성
conda create -n 3dgs python=3.8 -y

# 2. PyTorch cu118
conda run -n 3dgs pip install torch==2.0.1+cu118 torchvision==0.15.2+cu118 \
  --index-url https://download.pytorch.org/whl/cu118

# 3. GCC 11 (CUDA 11.8 컴파일러 호환)
conda install -n 3dgs -c conda-forge gcc=11 gxx=11 -y

# 4. 의존성
conda install -n 3dgs -c conda-forge libxcrypt -y
conda run -n 3dgs pip install plyfile tqdm pillow opencv-python-headless scipy scikit-learn

# 5. 소스 클론
cd /path/to/engine-vision
git clone --recursive https://github.com/graphdeco-inria/gaussian-splatting
cd gaussian-splatting

# 6. submodule 빌드 (WSL2 링커 경로 수동 지정 필요)
LDFLAGS="-L/lib/x86_64-linux-gnu -Wl,-rpath,/lib/x86_64-linux-gnu" \
TORCH_CUDA_ARCH_LIST="8.9" \
CPATH="/home/$USER/anaconda3/envs/3dgs/include:$CPATH" \
conda run -n 3dgs pip install \
  submodules/diff-gaussian-rasterization \
  submodules/simple-knn \
  submodules/fused-ssim

# 7. COLMAP (CPU 버전)
conda install -n 3dgs -c conda-forge "colmap=3.13.0=cpu*" -y
```
